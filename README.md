# BaseFramework

**BaseFramework** es una API RESTful desarrollada con Spring Boot que proporciona una base sólida para la gestión de usuarios con autenticación y autorización basada en roles (RBAC).

## ✨ Características

- Gestión completa de **usuarios** (creación, actualización, eliminación)
- Sistema de **roles** para control de acceso
- **Permisos granulares** por acción y recurso
- Organización por **módulos** funcionales
- Autenticación mediante JWT
- Encriptación de contraseñas
- Persistencia con JPA / MySQL
- Configuración de seguridad personalizable

## 🎯 Propósito

Este proyecto sirve como punto de partida para cualquier aplicación que requiera seguridad y organización de usuarios. Su diseño modular permite adaptarlo fácilmente a diferentes dominios mediante la extensión de entidades y endpoints.

## 🛠️ Tecnologías

- Java 17+
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- MySQL
